# TaskDuty Frontend

A task management web application built with React, TypeScript, and TailwindCSS.

## Tech Stack
- React + TypeScript
- TailwindCSS
- React Router DOM
- Vite

## Getting Started

### Install dependencies
npm install

### Run development server
npm run dev

## Features
- View all tasks
- Create new tasks
- Edit existing tasks
- Delete tasks
- Filter tasks by category
